## 📝 学習メモ

### 覚えたコマンド

```bash
# リポジトリの初期化
git init

# 変更をステージング
git add .

# コミット
git commit -m "メッセージ"

# GitHub にプッシュ
git push

# ブランチ作成と切り替え
git checkout -b ブランチ名

# ブランチ切り替え
git checkout ブランチ名
```
